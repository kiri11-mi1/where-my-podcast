<h1>Как установить?</h1>

  1. Установите интерпретатор Python3
  2. Скачайте pipenv
  3. Загрузите репозиторий
  4. Перейдите в корневую директорию загруженного репозитория
  5. Напишите команду ```pipenv install --dev```
  6. Присвойте все переменные локального окружения, которые нужны в коде
  7. Установите утилиту ffmpeg или acconv:
     * Для Unix систем ```sudo apt-get install ffmpeg```
     * Для Windows нужно скачать [архив](https://drive.google.com/file/d/1YZDHcEGNZABcYoVaxP4yxbARU7ldEag8/view?usp=sharing) и распаковать содержимое(не забудьте установить youtube-dl.exe в [переменную среды PATH](https://youtu.be/d7DJkiLKat0?t=71))
  8. Запустите программу командой ```python main.py```
